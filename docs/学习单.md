## 判断正比例的"三步法"

```
┌─────────────────────────────────────┐
│  第一步：两种量是否相关联？          │
│         ↓                          │
│  第二步：一种变化，另一种是否变化？  │
│         ↓                          │
│  第三步：比值是否一定？              │
│         ↓                          │
│  三步都满足 → 成正比例 ✓            │
│  任一步不满足 → 不成正比例 ✗        │
└─────────────────────────────────────┘
```
