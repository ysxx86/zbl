## 二、课前互动设计（5分钟）

### 主题：「星际旅行计划」

#### 设计理念
- **核心概念铺垫**：让学生在正式学习前，直观感受"速度一定时，距离与时间的关系"
- **情境延续**：与主课件的宇宙探索主题一致
- **互动形式**：全班参与，教师操作，学生举手/回答

#### 互动流程

```
┌─────────────────────────────────────────────────────────┐
│  课前互动：星际旅行计划（5分钟）                           │
├─────────────────────────────────────────────────────────┤
│  【第1步】情境引入（1分钟）                               │
│  - 展示太阳系行星图                                       │
│  - 提问："如果我们要去星际旅行，最想知道什么？"            │
│  - 引出：距离、时间、速度                                 │
├─────────────────────────────────────────────────────────┤
│  【第2步】速度大猜想（1.5分钟）                           │
│  - 展示飞船速度：10万千米/小时                            │
│  - 展示目的地距离（月球、火星、木星）                      │
│  - 学生举手猜测需要多少小时                               │
│  - 点击揭示答案，配合音效                                 │
├─────────────────────────────────────────────────────────┤
│  【第3步】规律发现（1.5分钟）                             │
│  - 展示表格：目的地 | 距离 | 时间                        │
│  - 引导学生发现：距离÷时间 = 10（都是10万千米/小时）       │
│  - 提问："你们发现了什么秘密？"                           │
│  - 预设答案：速度不变，距离越远时间越长                    │
├─────────────────────────────────────────────────────────┤
│  【第4步】过渡引入（1分钟）                               │
│  - 总结：距离和时间之间有神奇的关系                       │
│  - 引入正课："让我们启动飞船，深入探索这个秘密！"          │
│  - 点击按钮，火箭发射动画，进入Slide1                     │
└─────────────────────────────────────────────────────────┘
```

#### 数据设计
| 目的地 | 距离（万千米） | 时间（小时） | 比值 |
|--------|---------------|-------------|------|
| 月球 | 38 | 3.8 | 10 |
| 火星 | 5500 | 550 | 10 |
| 木星 | 63000 | 6300 | 10 |

## 四、教案设计

### 教案基本信息

```
┌─────────────────────────────────────────────────────────┐
│                    《正比例》教案                         │
├─────────────────────────────────────────────────────────┤
│  学科：数学                                              │
│  年级：六年级下册                                        │
│  教材版本：北师大版                                      │
│  单元：第四单元《正比例和反比例》                         │
│  课时：第1课时（共2课时）                                │
│  课型：新授课                                            │
│  授课时长：40分钟 + 课前互动5分钟                        │
└─────────────────────────────────────────────────────────┘
```

### 一、教学目标

#### 知识与技能
1. 结合具体情境，理解正比例的意义
2. 能判断两种量是否成正比例
3. 能找出生活中成正比例的例子

#### 过程与方法
1. 经历"情境探索→数据分析→规律发现→概念抽象"的学习过程
2. 体验从具体到抽象的数学思维方法

#### 情感态度与价值观
1. 感受数学与生活的紧密联系
2. 培养观察、分析、归纳的能力
3. 激发对数学学习的兴趣

### 二、教学重难点

| 类型 | 内容 | 突破策略 |
|------|------|---------|
| **重点** | 理解正比例的意义 | 通过多个实例，引导学生发现"比值一定"的本质特征 |
| **难点** | 判断两种量是否成正比例 | 设计正反例对比，建立"三步判断法" |

### 三、教学准备

| 类型 | 内容 |
|------|------|
| 教师准备 | 互动课件、教案、学习单（每人一份） |
| 学生准备 | 铅笔、橡皮、直尺 |

#### 【概念学习】揭开"正比例"的神秘面纱（5分钟）

```
教学环节：概念抽象
─────────────────────────────────────────────
教师活动：
1. 打开Slide4，点击"提取核心规律"按钮
2. 展示正比例的定义
3. 讲解："两种相关联的量，一种量变化，另一种量也随着变化。
   如果这两种量中相对应的两个数的比值一定，这两种量就叫做
   成正比例的量，它们的关系叫做正比例关系。"
4. 板书公式：y ÷ x = k（一定）
5. 组织拖拽分类游戏，判断哪些是正比例

学生活动：
1. 认真听讲，理解概念
2. 参与拖拽分类游戏
3. 在学习单上记录正比例的定义

学习单任务：
抄写正比例的定义和公式

设计意图：
- 从具体到抽象，建立概念
- 通过游戏巩固理解
─────────────────────────────────────────────
```

## 六、教学角度改进建议

### 6.1 概念强化

| 改进点 | 当前状态 | 改进建议 |
|--------|---------|---------|
| 正比例判断步骤 | 隐含在练习中 | 增加"三步判断法"：①两种量是否相关联 ②一种变化另一种是否变化 ③比值是否一定 |
| 反例对比 | 有面积vs周长 | 可增加更多生活反例：年龄与身高、已走路程与剩余路程 |
| 比值强调 | 绿色显示 | 增加"不变"的文字提示或动画效果 |
