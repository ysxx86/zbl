# 项目配置

## gstack 配置
使用 gstack 的 /browse 技能进行网页浏览。

### 可用技能列表：
- /office-hours - 产品咨询，帮你理清思路
- /plan-ceo-review - 从 CEO 角度审视功能
- /plan-eng-review - 工程师经理审查架构
- /plan-design-review - 设计师审查 UI/UX
- /design-consultation - 设计咨询
- /review - 代码审查
- /ship - 发布代码
- /browse - 网页浏览
- /qa - 测试应用
- /qa-only - 仅测试
- /design-review - 设计审查
- /setup-browser-cookies - 设置浏览器 cookies
- /retro - 回顾统计
- /investigate - 调查问题
- /document-release - 生成发布文档
- /codex - 代码分析
- /careful - 谨慎执行模式
- /freeze - 冻结状态
- /guard - 保护模式
- /unfreeze - 解除冻结
- /gstack-upgrade - 升级 gstack

如果 gstack 技能不工作，运行：
cd ~/.claude/skills/gstack && ./setup
